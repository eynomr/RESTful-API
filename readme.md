## How to use?
1. Close the repository
```bash
git clone 
```
2. navigate to folder
```bash
cd RESTful
```
3. Create and Activate Virtual Environment
```bash
conda create -n myenv
```
4. Install dependencies
```bash
pip install -r requirements.txt
```
5. Run the application
```bash
flask run
```
